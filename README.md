# RescueOS — Live-образ для системного администратора и реаниматора

**RescueOS** — это минималистичный Live-образ на базе Ubuntu 24.04 LTS (Noble), предназначенный для диагностики, восстановления системы и настройки сетевого оборудования. Образ полностью настраивается через bash-скрипт сборки, поддерживает русскую локализацию, статическую сеть с несколькими IP, встроенный DHCP-сервер (dnsmasq), предустановленные сетевые утилиты и Python-пакеты для автоматизации.

> ⚠️ **Внимание:** образ собирается в окружении обычного пользователя с правами `sudo`. После сборки вы получаете ISO-файл, который можно записать на флешку и загрузить на любом x86_64-совместимом компьютере.

## 📦 Что входит в образ

### Системные утилиты
- `aria2`, `curl`, `wget`, `ftp`, `rsync` – загрузка и синхронизация
- `netcat-openbsd` (`nc`), `socat` – сетевая отладка
- `openssh-server` – удалённый доступ
- `parted`, `fdisk`, `e2fsprogs`, `dosfstools` – работа с разделами
- `smartmontools` (`smartctl`) – диагностика дисков
- `grub-pc`, `grub-efi`, `os-prober` – восстановление загрузчика
- `iwd`, `systemd-networkd` – современное управление сетью
- `far2l` – файловый менеджер (Far Manager clone)
- `nano`, `less`, `jq`, `sqlite3` – работа с текстом, JSON, базами
- `python3`, `pip3` и предустановленные модули: `requests`, `paramiko`, `psutil`

### Файловые системы Windows
- Поддержка **NTFS** (`ntfs-3g`, `ntfs3`), **exFAT** (`exfatprogs`), **FAT32** (`dosfstools`)

### Сеть
- Статическая настройка с **двумя IP-адресами** на одном интерфейсе (по умолчанию `192.168.137.110/24` и `192.168.0.110/24`)
- Готовый конфиг для **DHCP-сервера (dnsmasq)** для подсети `192.168.137.0/24` (запуск вручную)
- Подсказки по добавлению дополнительных IP и маршрутов в приветственном сообщении

### Пользователи и локализация
- Имя хоста: `rescuebox`
- Основной пользователь: `rescue` / пароль `rescue`
- Root: `rescue`
- Русская консоль (`ru_RU.UTF-8`), шрифт `cyr-sun16`, раскладка `us,ru`

### Кастомизация
- Все пользовательские файлы из `~/my_custom_files` копируются в `/home/rescue/` в образе
- Возможность легко добавить свои скрипты, конфиги или утилиты

## 🛠️ Требования к хост-системе для сборки

- **ОС:** Ubuntu 22.04 / 24.04 (или Debian 12+ с установленным `bash`)
- **Пользователь:** обычный, с доступом `sudo`
- **Свободное место:** минимум 8 ГБ
- **Интернет:** для загрузки пакетов (только во время сборки)
- **Пакеты:** будут автоматически установлены скриптом (`debootstrap`, `xorriso`, `squashfs-tools`, `grub-*`, `mtools`, `aria2`)

## 📥 Сборка образа

	1. **Скачайте скрипт** `build.sh` и сделайте его исполняемым:
   ```bash
   chmod +x build.sh
	2. Отредактируйте переменные в начале файла (по желанию):
    RELEASE, ARCH, MIRROR
    HOSTNAME, USERNAME, USER_PASS, ROOT_PASS
    NETWORK_MODE (static/dhcp), STATIC_IPS, STATIC_GATEWAY, STATIC_DNS
    INSTALL_DHCP_SERVER (yes/no)
    EXTRA_SYSTEM_PACKAGES, EXTRA_PIP_PACKAGES
    CUSTOM_FILES_DIR (по умолчанию ~/my_custom_files)
